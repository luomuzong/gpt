# GPT

This repository now includes a simple terminal Snake game implemented in `snake.py`.

## How to run the game

Make sure you have Python 3 installed. On Unix-like systems, you can start the game with:

```bash
python3 snake.py
```

Use the arrow keys to control the snake. Eat the `*` food to grow longer and gain points. The game ends when the snake hits the wall or itself.
